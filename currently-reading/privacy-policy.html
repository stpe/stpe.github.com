Currently Reading (Chrome Extension) - Privacy Policy

  • Currently Reading only commmunicates with the Literal API, using the username you provide to retrive reading data. No password is asked for, everything is public.
  • The only data stored, in your browser using the Chrome Storage API (which only your Chrome user can access), is your Literal username and associated ID. This is to avoid having to ask for your username every time you use the extension.
  • There is no usage tracking or analytics data collected, sent or stored.
